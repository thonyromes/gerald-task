## :rocket: Installation
1. Get the code

    ```bash
    git clone git@github.com:thonyromes/gerald-task.git && cd gerald-task
    ```

2. Install expo-cli

    ```bash
    npm install -g expo-cli
    ```

3. Install dependencies

    ```bash
    yarn
    ```

4. In terminal, run

    ```bash
    expo start
    ```
    
5. Commands in running expo terminal
  
   ```bash
    'a' to run in Android
   ```
